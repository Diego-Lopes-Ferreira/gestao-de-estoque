# Gestao de Estoque

## Development
```sh
yarn dev # start the development server

yarn prisma init # Set up Prisma for your app
yarn prisma generate # Generate artifacts (e.g. Prisma Client)
yarn prisma db # Manage your database schema and lifecycle
yarn prisma migrate # Migrate your database
yarn prisma studio # Browse your data with Prisma Studio
yarn prisma validate # Validate your Prisma schema
yarn prisma format # Format your Prisma schema
```
